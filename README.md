# 🏧 ATM Management System

A simple **Python Command Line Interface (CLI)** project that simulates basic ATM operations. This project is developed using Python modules, functions, and user-defined exception handling.

---

## 📌 Features

- 💰 Deposit Money
- 💸 Withdraw Money
- 💳 Balance Enquiry
- 🚪 Exit the Application
- ⚠️ User-defined Exception Handling
- 📂 Modular Programming

---

## 🛠 Technologies Used

- Python 3
- Functions
- Modules
- Exception Handling
- User-defined Exceptions
- Match-Case Statement

---

## 📁 Project Structure

```
ATM-Management-System/
│
├── ATMmainprogram.py      # Main program
├── ATMoperation.py        # Deposit, Withdraw, Balance Enquiry
├── ATMMenu.py             # Displays ATM Menu
├── ATMExcept.py           # User-defined Exceptions
└── README.md
```

---

## ▶️ How to Run

1. Clone the repository.

```bash
git clone https://github.com/Jayanthi740/ATM-Management-System.git
```

2. Open the project folder.

```bash
cd ATM-Management-System
```

3. Run the project.

```bash
python ATMmainprogram.py
```

## ⚙️ Exception Handling

The project uses custom exceptions to validate user input.

- DepositError
- WithDrawError
- InsufFundError
- ValueError

These exceptions help prevent invalid operations and improve the reliability of the application.
